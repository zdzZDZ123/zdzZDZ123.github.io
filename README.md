# 手势控制的 3D 星空

使用 Three.js 与 MediaPipe Hands 构建的交互式 3D 星空体验。通过摄像头识别手势即可在浏览器中旋转、缩放并选中星体。

## 功能特性

- **沉浸式 3D 场景**：随机生成的 3D 星空，包含光晕恒星与环绕行星。
- **详细信息展示**：选中星体时显示类型、温度/重力、质量/大气等详细数据。
- **自定义设置**：内置设置面板，可实时调整旋转速度和手势灵敏度。
- **手势交互系统**：
  - **张开手掌**：平滑拉远镜头（Zoom Out）
  - **握拳**：平滑推进镜头（Zoom In）
  - **食指指向**：精确选中星体查看详情
  - **手掌移动**：全方位旋转观察视角
- **高性能**：优化的渲染循环与手势检测逻辑，支持配置参数调整。

## 快速开始

1. 在本地启动任意静态文件服务（需支持 ES Modules）：

   ```bash
   npx serve .
   # 或
   python3 -m http.server 4173
   ```

2. 使用现代浏览器（建议 Chrome 113+ 或 Edge 113+）访问启动的地址。

3. 允许浏览器访问摄像头，即可看到手势控制提示与实时视频画面。

## 手势控制指南

- **张开手掌 (Open)**：
  - 所有手指伸直并张开，用于拉远视角，概览星空。
- **握拳 (Fist)**：
  - 手指收拢握拳，用于推进视角，观察细节。
- **指向 (Point)**：
  - 食指伸直，其余手指收拢（拇指可自然放置）。指向屏幕上的星体进行选中。
- **自然移动**：
  - 手掌在画面中的位置决定了视角的旋转方向。

## 优化与改进

本项目在原基础上进行了以下优化：
- **代码重构**：分离配置参数至 `config.js`，减少魔法数字，提高可维护性。
- **功能增强**：增加了星体详细信息（温度、质量等）展示。
- **用户体验**：添加了设置面板，允许用户调整灵敏度和旋转速度。
- **算法优化**：改进了手势判定逻辑，增加了平滑处理，减少抖动。
