# 手势控制的 3D 星空

使用 Three.js 与 MediaPipe Hands 构建的交互式 3D 星空体验。通过摄像头识别手势即可在浏览器中旋转、缩放并选中星体。

## 功能特性

- 随机生成的 3D 星空场景，包含恒星与带光晕的行星
- 基于手势的交互：
  - **张开手掌**：缩小视角，观察更广阔的星空
  - **握拳**：拉近镜头，靠近星体
  - **指向**：使用食指选中、查看星体细节
  - **手掌移动**：旋转星空视角
- 选中星体时自动高亮并展示名称
- 全屏模式与响应式布局，支持桌面与移动端大屏浏览器

## 快速开始

1. 在本地启动任意静态文件服务（需支持 ES Modules）：

   ```bash
   npx serve .
   # 或
   python3 -m http.server 4173
   ```

2. 使用现代浏览器（建议 Chrome 113+ 或 Edge 113+）访问启动的地址。

3. 允许浏览器访问摄像头，即可看到手势控制提示与实时视频画面。

## 手势控制指南

- **张开手掌**：
  - 让所有手指伸直并张开手掌，镜头会平滑拉远以展示更大范围。
- **握拳**：
  - 将手指收拢形成拳头，镜头会靠近星体以便细节观察。
- **指向**：
  - 保持食指伸直，其余手指收拢。指向屏幕任意位置即可选中对应星体。
- **移动手掌**：
  - 以手掌为整体向左右或上下移动，星空视角会随之旋转。

## 技术栈

- **Three.js**：构建 3D 星空、材质与光照
- **MediaPipe Hands (Tasks Vision)**：实时手势识别
- **原生 ES Modules + HTML5 + CSS3**

## 注意事项

- 请确保使用具备 WebGL 与摄像头权限的现代浏览器。
- 初次进入页面需允许浏览器访问摄像头，否则手势识别无法启动。
- 若手势识别出现异常，可刷新页面或检查摄像头是否被其他应用占用。
